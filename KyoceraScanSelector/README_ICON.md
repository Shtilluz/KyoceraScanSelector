# Создание иконки для приложения

## Автоматическое создание

Для создания иконки запустите:

```bash
python create_icon.py
```

Это создаст файл `printer.ico` с иконкой принтера.

## Требования

- Python 3.6+
- Pillow (установится автоматически из requirements.txt)

```bash
pip install Pillow
```

## Ручное создание

Если скрипт не работает, вы можете создать иконку вручную:

1. Создайте изображение 256x256 пикселей с изображением принтера
2. Сохраните как `printer.ico`
3. Поместите файл в папку KyoceraScanSelector/

## Примечания

- Иконка используется в заголовке окна приложения
- При сборке через PyInstaller иконка встраивается в .exe файл
- Если иконка отсутствует, приложение будет работать с иконкой по умолчанию
